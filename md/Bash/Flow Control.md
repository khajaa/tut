
##Flow Control

###if statement - Example check string size
```bash
 if [ -z str ]; then
    echo "str is empty"
 else
    echo "str is not empty"
 fi
 ```

Example to change all .html files to .htm
```bash
 do
 mv $i ${i%.html}.htm
 done
 ```
Append icon_ in front of the finename

```bash
 do
 cp $i icon_$i
 done
 ```
