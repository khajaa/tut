|Description|Shortcut Keys|h
|Move to Code (Back from debug window)|F7|

|Move to next window|Ctrl + Tab or Ctrl + F7|
|Move to previous window|Ctrl + Shift + Tab|
|Move to Solution Explorer|Ctrl + Alt + L or Alt -> V -> P|
|Move to Property|F4|
|Close window|Ctrl + F4|
|Cycle Clipbord|Ctrl + Shift + V|
|Replace|Ctrl + H|
|Go to|Ctrl + G|
|Complete Word|Ctrl + Space|
|Delete Line|Shift + Del|
|Full Screen|Shift + Alt + Enter|
|Add New Item|Ctrl + Shift + A|
|Comment Out|(Select area)Ctrl + K, Ctrl + C|
|Remove Comment|(Select area)Ctrl + K, Ctrl + U|
|Mark|Ctrl + K, Ctrl + K|
