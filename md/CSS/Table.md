###Table Margin, Spacing, padding...
Remove all margin, spacing, etc...
    border-collapse: collapse;
    border-spacing:0px;

If you need a skinney table border, you have to put 
```css
 ```
e.g.
```css
 border-collapse: collapse;
 }
 #package table {
 /* .... */
 }
 #package td {
 /* .... */
 }
 ```
