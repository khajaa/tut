##Catch when visitor move from the page (onLoad, onUnload) ###Example ```javascript <HEAD> <TITLE>onLoad and onUnload example</TITLE> <SCRIPT TYPE="text/javascript"> <!-- function disp1(){ window.alert('Welcome'); } function disp2(){ window.alert('You are leaving'); } // --> </SCRIPT> </HEAD> <BODY onLoad="disp1()" onUnload="disp2()"> </BODY> </HTML> ```
