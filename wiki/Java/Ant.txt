***Installation
Download a zip file from
http://ant.apache.org/

Create dir ''c:\ant''

and unzip everything under this directory.

Add ''c:\ant\bin'' into ''PATH''
Set ''c:\ant'' to ''ANT_HOME''

Make sure ''JAVA_HOME'' is set to your ''c:\jdk???'' dirctory
